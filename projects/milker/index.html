The objective of this project was to design and develop a responsive website for a heavy machine parts retailer. The website aims to provide an intuitive user experience while showcasing the retailer's extensive range of products. Key features include a welcoming homepage, a product showcase, a contact form for inquiries, and a user-friendly mobile menu.
